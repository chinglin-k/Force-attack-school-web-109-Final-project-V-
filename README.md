## 專案簡介
使用Visual Studio 2019編譯<br>
#### 邏輯
1.使用C#編寫主程式進行身分證計算<br>
2.再呼叫Python編寫的副程式<br>
3.將正確的身分證字號輸入進副程式並執行登入<br>
4.若登入成功執行爬蟲；若失敗回傳'None'<br>
5.最後將參數傳回主程式<br>
6.若登入成功輸出爬蟲內容；若失敗繼續嘗試登入<br>
